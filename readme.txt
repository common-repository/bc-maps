This Plugins gives Buddypress and bettercodes a user, groups and Projects geo localisation feature

== Description ==

We developed this plugin primary for http://bettercodes.org/

* This is a unstable version of this plugin
* Please run test before use.
* Not recomended for productive invironment yet

== Installation ==

1. Upload `bc-maps` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress


== Frequently Asked Questions ==

= Do i need a google maps api key? =

No you don't. With the current google maps api version v3 you don't need a api key any more.
